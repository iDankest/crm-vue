# crm-vue


## Proyecto CRM para Vue

Este es un proyecto de CRM totalmente hecho con Vue.js, utilizando Vue Router para la navegación entre páginas y Vuex para el manejo de estados globales. Toda la documentación y detalles sobre cómo se creó este proyecto se encuentran en mi [Notion](https://www.notion.so/CRM-con-Routing-2715592c6f7e803fbbe8fa17973108ef?source=copy_link).

Para crear este proyecto, sigue estos pasos:

#### Crear un nuevo proyecto de Vue

```sh
npm create vite@latest crm-vue -- --template vue
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
